gulp-boilerplate
================

# Get Started
1. Clone this repository.
2. Remove these files.
```bash
rm -rf .git README.md
```
3. Install modules.
```bash
npm install
```
4. Launch your server.
```bash
gulp
```

Then you can access your app at http://localhost:3000/ :beer:

# Supporting Files
- Jade `.jade`
- CSS `.css`
- Stylus `.styl` `.stylus`
- JavaScript `.js`
- CoffeeScript `.coffee`
- HTML `.htm` `.html`
- Image `.jpg` `.png` `.gif`
